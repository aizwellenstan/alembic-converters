# Convert Alembic curve to CyHair format.

## Usage

```
$ abc2cyhair input.abc output.hair (flip_yz) (debug)
```

### Supported features.

* [x] Curve widths(exported as thickness in CyHair)
* [ ] Curve color
* [ ] Curve UVs
* [ ] Custom curve attributes
